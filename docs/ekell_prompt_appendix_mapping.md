# E-KELL Appendix A prompt mapping

These short templates reconstruct task structure, not Appendix A wording. They are
independent paraphrases and should not be described as official prompts.

- `query_decomposition.txt`: decomposition into the constrained logical-query AST.
- `logical_expression_validation.txt`: logical-form and KG-vocabulary validation.
- `stepwise_projection.txt`: Table 3 projection over an entity/relation pair.
- `stepwise_intersection.txt`: Table 3 intersection of operand answers.
- `stepwise_union.txt`: Table 3 union of operand answers.
- `stepwise_negation.txt`: Table 3 complement against an explicit universe.
- `final_kg_grounded_response.txt`: Appendix decision-support synthesis constrained to
  supplied KG triples, paths, and evidence IDs.
