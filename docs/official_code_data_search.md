# Official E-KELL Code/Data Search Status

Search target:

- Paper: E-KELL: Enhancing Emergency Decision-making with Knowledge Graphs and Large Language Models
- arXiv: https://arxiv.org/abs/2311.08732

Status at this stage:

**Official E-KELL code/data was not integrated in this repository at this stage.**

A public web/GitHub search during this upgrade did not identify an official E-KELL repository or official dataset that could be integrated directly. The arXiv page provides the paper and source links, but this repository does not treat those as official executable code/data for result reproduction.

Therefore, this project uses:

1. Public paper-level methodology descriptions.
2. Independently implemented KG loading, entity matching, subgraph retrieval, and prompt-chain modules.
3. Copied fire emergency input data files where available.
4. Explicit documentation of all deviations.

This file should be updated if official E-KELL code, data, prompts, or evaluation scripts become available and are verified.
